# Pennant Manager DX

**Pennant Manager DX** は、ブラウザで遊べる野球監督シミュレーションゲームです。

プレイヤーは **TOKYO BLUE STARS** の監督となり、  
サイン、投球モード、継投、選手特性、投手スタミナを見極めながら、  
ペナントレースを勝ち抜いて日本一を目指します。

## 🎮 ゲーム概要

このゲームでは、プレイヤー自身が直接選手を操作するのではなく、  
監督として試合状況を読み、采配を行います。

カウント、走者、守備力、投手スタミナ、選手の特性などを確認しながら、  
場面に応じて最適な指示を出すことが勝利へのポイントです。

## ⚾ 主な特徴

- ブラウザだけで遊べる野球監督ゲーム
- 守備時の采配
  - 勝負
  - 一球外す
  - 申告敬遠
  - 牽制
- 投球モード切替
  - 球種：オート
  - 球種：手動
- 球種指定機能
- 投手スタミナ管理
- 継投システム
- 選手特性
  - エース
  - ホームランバッター
  - 盗塁王
  - 強肩
  - 代打の切り札
- リーグ順位表
- 選手成績・ベンチ表示
- JSON形式でのセーブ / ロード
- ダーク / ライト / クリアーテーマ対応
- 初回チュートリアル
- デモンストレーション表示

## 🕹️ 遊び方

1. ページを開く
2. 初回チュートリアルを確認
3. 先発投手を選択
4. 試合状況を見ながら采配
5. 勝利を重ねてリーグ上位を目指す

守備時は、相手打者や投手スタミナを見ながら、  
「勝負」「一球外す」「申告敬遠」「牽制」などを選びます。

投球モードを **球種：手動** にすると、  
投手が持つ球種を1球ごとに指定できます。

## 💾 セーブ / ロード

左側の **Save / Load** から、現在のゲームデータをJSON形式で保存できます。

- `JSONをダウンロード（SAVE）`
- `JSONを読み込む（LOAD）`

ブラウザを閉じる前や、続きから遊びたい場合に利用してください。
